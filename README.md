# AI Site Studio

A Lovable.dev-inspired AI platform UI scaffold built with React, TypeScript, Vite, Tailwind, and Supabase.

## Features
- Lovable-style layout, command palette, and builder tabs
- Supabase schema & edge function stubs
- Credits gating, onboarding, and empty states

## Local development
```bash
npm install
npm run dev
```

## Supabase
- Apply migrations from `supabase/migrations`.
- Deploy the `ai-chat` edge function.

## Environment
Copy `.env.example` to `.env` and fill in your Supabase keys.
